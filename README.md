# Artin Heidari — Portfolio

A recruiter-focused engineering portfolio built with Astro, handwritten CSS, and Markdown content collections.

## Current status

V1 contains one clearly labelled sample project and one sample blog post to preview the content system. They must be replaced with real work before the public launch.

## Local development

```bash
npm install
npm run dev
```

Astro serves the site locally and reloads it when files change.

## Production check

```bash
npm run build
npm run preview
```

The production output is generated in `dist/`.

## Content

- Project case studies: `src/content/projects/`
- Blog posts: `src/content/blog/`
- Shared styling: `src/styles/global.css`
- Page components: `src/components/`
